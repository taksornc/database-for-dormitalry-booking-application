# database-for-dormitalry-booking-application
ออกแบบโครงสร้างฐานข้อมูลด้วย ER Diagram และ Relation Schema สำหรับระบบจองหอพัก
พัฒนาและจัดการฐานข้อมูลด้วย MySQL โดยใช้ phpMyAdmin บน XAMPP
เชื่อมต่อฐานข้อมูลกับระบบด้วย PHP รองรับการทำงานทั้งฝั่งผู้จองและผู้ดูแลระบบ 
จัดการข้อมูลห้องพัก ผู้เช่า การจอง และการชำระเงิน
ทำ Normalization (1NF–3NF) เพื่อลดความซ้ำซ้อนและเพิ่มประสิทธิภาพข้อมูล
